Rules of the Snowbot Rush

Rule 1:

How to Play Snowbot Rush

Step 1: Enter name to start game

Step 2: Wait for other players to join

Step 3: When everyone joins, the game board would display to players

Step 4: Players can click any box they want in the game board

Step 5: If the box the player clicked is a snowflake or a snow enemy, the player would gain or loose a life respectively. If the box the player clicked is a coin, jewel, or a diamond, the player would gain 10, 25, 50 points respectively. If the box the player clicked is ice, the player would be frozen for 3 seconds, which means that the player can not play for the next 3 seconds of the game. If the box the player clicked is a X, the player gets notified that the other player has claimed the object in the box.
